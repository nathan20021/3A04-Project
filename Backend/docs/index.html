POST Register
http://localhost:3000/users/register

Register endpoint. Pass a username, email, password in body and user is registered. Returns user_id in response which can be passed to pages on frontend through props.

BODY raw
                                                  
                                                    
{
    "username": "areez_visram",
    "password": "password123",
    "email": "areez.visram10@gmail.com"
}
Example Request
  • User Registration Example
                                                
POST http://localhost:3000/users/register
{
    "username": "areez_visram",
    "password": "password123",
    "email": "areez.visram10@gmail.com"
}
Example Response
200 - OK 
                                                    
{
    "status": "success",
    "user_id": 6
}
POST Login
http://localhost:3000/users/login

Login endpoint. Pass a username and password in body and it checks if user exists. If it does, user is logged in and user_id included in response.

BODY raw
                                                  
                                                    
{
    "username": "areez_visram",
    "password": "password123"
}
Example Request
  • User Login Example
                                                
POST http://localhost:3000/users/login
{
    "username": "areez_visram",
    "password": "password123"
}
Example Response
200 - OK 
                                                    
{
    "status": "success",
    "user_id": 1
}
POST Delete User
http://localhost:3000/users/delete

Delete user endpoint. Pass a user_id and if a user with that ID exists, it is deleted from the database.

BODY raw

    "user_id": 6
}
Example Request
  • Delete User Example
                                                
POST http://localhost:3000/users/delete
{
    "user_id": 6
}
Example Response
200 - OK 
                                                    
{
    "status": "success"
}
POST Edit User
http://localhost:3000/users/edit

Edit user. Pass a username, email, password and userid and if a user with matching userid exists in the database, their info will be updated with the passed parameters.

BODY raw
                                                  
                                                    
{
    "username": "acheese",
    "email": "areez.visram1010@gmail.com",
    "password": "bruh",
    "user_id": 5
}
Example Request
  • Edit User Example
                                                
POST http://localhost:3000/users/edit
{
    "username": "acheese",
    "email": "areez.visram1010@gmail.com",
    "password": "bruh",
    "user_id": 5
}
Example Response
400 - Bad Request 
                                                    
{
    "status": "error",
    "message": "The specified user does not exist"
}
